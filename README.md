# FocusFlow Landing Page

[![Live Demo](https://img.shields.io/badge/Live%20Demo-View%20Website-blue?style=for-the-badge)]()

A modern, responsive landing page built for a conceptual AI-powered workspace app called FocusFlow.

## Live Interaction & Functionality

While this is a static landing page, it includes interactive UI components that simulate the feel of a real application. You can explore and try it out directly in your browser. 

**Key interactive elements you can test:**
- **Dashboard Simulation:** In the hero section, click through the dashboard tabs (Today, This week, Backlog) to see how the tasks dynamically filter and change.
- **Interactive FAQ:** Click on the questions in the FAQ section to see smooth accordion dropdown animations revealing the answers.
- **Dynamic Pricing Toggle:** Switch between Monthly and Yearly pricing tiers to see the numbers update instantly.
- **Mobile Menu:** Resize your browser to mobile width and try the custom animated hamburger dropdown menu.

## About the Project

I built this project to practice modern web design, complex CSS layouts, and micro-interactions.
- **Advanced UI Mockup:** The dashboard featured in the hero section is not a static image—it's a fully coded, structured UI component built from scratch using HTML and CSS (Flexbox/Grid) to demonstrate advanced layout skills.
- **Rich Aesthetics:** Implements glassmorphism (frosted glass effects), dynamic aurora gradients and subtle hover animations to create a premium feel.
- **Fully Responsive:** Carefully adapted to look and behave perfectly across mobile devices, tablets and full desktop displays.
- **Clean Code:** Built entirely from the ground up without heavy UI libraries (like Bootstrap or Tailwind) to ensure maximum control over the styling and animations.

## Tech Stack

- **HTML5:** Semantic structure and organization.
- **CSS3:** Custom properties (variables), Flexbox, Grid, advanced keyframe animations, and backdrop-filters.
- **JavaScript:** Used purely for DOM manipulation to drive the interactive dashboard tabs, FAQ accordions, pricing logic, and scroll progress tracking.

## How to Run

1. Clone or download this project.
2. Open the `index.html` file in any modern web browser.
3. No servers, build tools, or installations are required!
